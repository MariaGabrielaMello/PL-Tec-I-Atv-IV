## Guia de execução no terminal

- git clone https://github.com/MariaGabrielaMello/PL-Tec-I-Atv-IV.git

Após clonar o projeto, executar no terminal:

### Backend:
- cd atviv
- cd executavel
- java -jar pl.jar
  
### Frontend:
- cd atviv
- cd src
- npm install
- npm start
